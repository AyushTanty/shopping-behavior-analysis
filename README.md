# 🛍️ Customer Shopping Behavior Analysis

This project focuses on analyzing customer shopping data to extract meaningful insights using data cleaning, processing, and visualization techniques.

## 🚀 Key Features
- Data cleaning (handling missing values, duplicates, and outliers)
- Exploratory Data Analysis (EDA)
- Visualization using Matplotlib
- Insight generation and storytelling

## 📊 Key Insights
- `Clothing` category generates the highest revenue (~46.6%)
- `Sandals` have the highest average customer rating
- `Shirt` is the most frequently purchased item but has lower satisfaction
- Majority of buyers (`~65%`) are male customers
- `Black` is the most preferred color across customers

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📌 Conclusion
This analysis helps understand customer preferences, top-performing products, and revenue-driving categories, which can support better business decisions.
